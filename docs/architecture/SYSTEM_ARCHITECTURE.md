# WoodFlow System Architecture

## Overview

WoodFlow is an offline-first Progressive Web Application (PWA) designed for woodworking businesses.

The system consists of several core modules:

- User Interface
- Material Management
- Panel Inventory
- Offcut Inventory
- Project Management
- Cutting Engine
- Visualization Engine
- Local Database

This document will evolve together with the project and describes the overall software architecture.
