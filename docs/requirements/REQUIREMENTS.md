# WoodFlow Requirements

## Project Goal

WoodFlow is an offline-first Progressive Web Application that helps woodworking businesses manage panel inventory, reusable offcuts and cutting layouts.

The primary goal is to reduce material waste while simplifying workshop workflows.

---

# Functional Requirements

## Inventory

- Store full panels
- Store reusable offcuts
- Manage quantities
- Manage materials
- Manage panel thickness
- Edit and delete inventory

## Projects

- Create woodworking projects
- Manage required parts
- Assign materials
- Save project history

## Cutting

- Calculate cutting layouts
- Support configurable kerf width
- Support optional part rotation
- Generate reusable offcuts automatically

## Visualization

- Display cutting layouts
- Display dimensions
- Display remaining offcuts
- Provide printable layouts

## QR Support

- Generate QR codes
- Scan QR codes
- Identify inventory quickly

## Export

- Export cutting layouts as PDF
- Export inventory reports

---

# Non-Functional Requirements

- Offline First
- Responsive Design
- Local Data Storage
- Millimeter Precision
- Fast User Interface
- Easy Workshop Usage
