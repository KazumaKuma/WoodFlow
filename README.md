#  WoodFlow

> Simple inventory. Smarter cutting. Less waste.

WoodFlow helps small woodworking businesses manage panels, reusable offcuts and cutting workflows.

WoodFlow is optimized for simplicity, affordability and everyday workshop workflows instead of enterprise-level production management.

---

## Planned Features

-  Panel Inventory
-  Offcut Management
-  Project Management
-  Cutting Optimization
-  Visual Cutting Layouts
-  Progressive Web App
-  Offline First

## About

WoodFlow started as a custom software project for a local woodworking business.

Instead of replacing enterprise ERP systems, WoodFlow focuses on solving everyday workshop problems through a simple, offline-first workflow.

The project is developed incrementally together with real user feedback.
